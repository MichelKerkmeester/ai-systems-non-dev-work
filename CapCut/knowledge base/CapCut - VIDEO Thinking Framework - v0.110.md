## 1. üéØ VIDEO PRINCIPLES

### The VIDEO Methodology

**V**isualize ‚Üí **I**nvestigate ‚Üí **D**esign ‚Üí **E**xecute ‚Üí **O**utput

A systematic 5-phase approach to video editing automation that ensures comprehensive analysis and optimal output.

### Phase Breakdown

**1. VISUALIZE (V)**
- Understand the user's creative vision
- Identify target platform and audience
- Determine style and aesthetic requirements
- Clarify duration and pacing expectations

**2. INVESTIGATE (I)**
- Analyze source media specifications
- Check format compatibility
- Verify file accessibility and quality
- Identify technical constraints

**3. DESIGN (D)**
- Plan timeline structure
- Select appropriate tracks
- Choose transitions and effects
- Map text and audio placement

**4. EXECUTE (E)**
- Create draft with optimal settings
- Apply segments to tracks
- Add animations and effects
- Process with quality controls

**5. OUTPUT (O)**
- Export to JianYing format
- Validate project structure
- Provide user guidance
- Suggest refinements

### Two-Layer Transparency

**Internal Processing (Full Depth):**
- Complete VIDEO methodology application
- Resolution and framerate analysis
- Platform compatibility assessment
- Effect availability verification
- Timeline optimization logic

**External Communication (Concise):**
- Progress updates by phase
- Key decisions explained
- Quality confirmations
- Next steps suggested

## 3. üìä QUALITY OPTIMIZATION

### Resolution Guidelines

| Use Case              | Resolution    | Aspect Ratio | Notes                  |
| --------------------- | ------------- | ------------ | ---------------------- |
| YouTube/Web           | 1920x1080     | 16:9         | Standard HD            |
| TikTok/Reels/Shorts   | 1080x1920     | 9:16         | Vertical mobile        |
| Instagram Feed        | 1080x1080     | 1:1          | Square format          |
| Instagram Story       | 1080x1920     | 9:16         | Full screen vertical   |
| Professional/4K       | 3840x2160     | 16:9         | High quality           |
| Twitter/X             | 1280x720      | 16:9         | Optimized for platform |

### Framerate Selection

| Content Type        | Framerate | Reasoning                     |
| ------------------- | --------- | ----------------------------- |
| Standard content    | 30 fps    | Universal compatibility       |
| Smooth motion       | 60 fps    | Gaming, sports, fast action   |
| Cinematic look      | 24 fps    | Film aesthetic                |
| PAL broadcast       | 25 fps    | European standard             |
| Slow motion source  | Match     | Preserve original quality     |

### Quality vs Size Trade-offs

**High Quality Priority:**
- Archive/master copies
- Professional delivery
- Large screen display
- Print/broadcast

**Balanced (Recommended):**
- Web streaming
- Social media
- General sharing
- Mobile viewing

**Size Priority:**
- Email attachments
- Limited bandwidth
- Quick previews
- Mobile data

### Platform-Specific Recommendations

**YouTube:**
```yaml
youtube:
  resolution: 1920x1080 (or 3840x2160 for 4K)
  framerate: 30 or 60 fps
  aspect_ratio: 16:9
  max_duration: 12 hours
  recommended:
    shorts: 1080x1920, 9:16, <60s
    standard: 1920x1080, 16:9
```

**TikTok:**
```yaml
tiktok:
  resolution: 1080x1920
  framerate: 30 fps
  aspect_ratio: 9:16
  max_duration: 10 minutes
  optimal_duration: 15-60 seconds
```

**Instagram:**
```yaml
instagram:
  feed:
    resolution: 1080x1080 (square) or 1080x1350 (4:5)
    aspect_ratio: 1:1 or 4:5
    max_duration: 60 seconds
  reels:
    resolution: 1080x1920
    aspect_ratio: 9:16
    max_duration: 90 seconds
  stories:
    resolution: 1080x1920
    aspect_ratio: 9:16
    max_duration: 60 seconds
```

### Effect Categories

**Video Animations:**
- Entrance: fade_in, slide_in, zoom_in, rotate_in
- Exit: fade_out, slide_out, zoom_out, rotate_out
- Loop: pulse, shake, bounce

**Transitions:**
- Basic: dissolve, fade, cut
- Motion: slide, wipe, zoom
- Creative: blur, distort, glitch

**Filters:**
- Color: warm, cool, vibrant, muted
- Style: vintage, modern, cinematic
- Mood: bright, dark, dramatic

    - track_0: background_music

### Timeline Best Practices

- Use milliseconds for precision
- Account for transition overlaps
- Maintain consistent spacing
- Align to beat for music

**Effect Application:**
- Apply animations after segments
- Set transition duration appropriately
- Layer effects intentionally
- Test keyframe smoothness

## 6. üõ†Ô∏è MCP TROUBLESHOOTING

### Connection Diagnostics

**Verification Steps:**
1. Check MCP server is running
2. Verify uv environment active
3. Confirm SAVE_PATH and OUTPUT_PATH set
4. Test with `rules` command

**Common Issues:**

```yaml
connection_issues:
  server_not_running:
    symptom: "Connection refused"
    solution: "Start server with uv run server.py"

  environment_not_set:
    symptom: "SAVE_PATH/OUTPUT_PATH undefined"
    solution: "Export environment variables"

  python_version:
    symptom: "Python version error"
    solution: "Ensure Python 3.13+ installed"

  uv_not_found:
    symptom: "uv command not found"
    solution: "Install uv package manager"
```

### JianYing App Issues

**Desktop App Requirements:**
- JianYing Pro must be installed
- App should be running for draft import
- Drafts appear in JianYing's draft list

**Troubleshooting:**

```yaml
app_issues:
  draft_not_appearing:
    symptom: "Exported draft not visible in JianYing"
    solutions:
      - "Verify OUTPUT_PATH matches JianYing draft folder"
      - "Restart JianYing Pro"
      - "Check file permissions"

  import_failed:
    symptom: "JianYing cannot open draft"
    solutions:
      - "Verify draft structure integrity"
      - "Check for missing media files"
      - "Re-export draft"
```

### Path Configuration
