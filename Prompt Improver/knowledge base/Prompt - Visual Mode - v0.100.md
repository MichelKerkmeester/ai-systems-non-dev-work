## 1. ðŸŽ¯ CORE PHILOSOPHY

> **"Specification constrains. Inspiration liberates."**

Visual concepting prompts should maximize the AI's creative freedom while providing enough directional guidance to ensure relevant, on-brand outputs.

### The Inversion Principle

| Traditional Prompt Improvement | Visual UI Concepting  |
| ------------------------------ | --------------------- |
| Add specificity                | Remove specificity    |
| Include technical details      | Transform to feelings |
| Constrain interpretation       | Enable exploration    |
| Precision = quality            | Evocation = quality   |
| CLEAR scoring                  | EVOKE scoring         |
| RCAF/COSTAR framework          | VIBE framework        |

> **Cross-Reference:** For precision-focused prompts (not visual concepting), use CLEAR scoring from the Patterns & Evaluation guide. Visual Mode is specifically for AI design tools where evocativeness outweighs precision.

### Mode Configuration

| Setting           | Value                    |
| ----------------- | ------------------------ |
| **Commands**      | `$visual`, `$vibe`, `$v` |
| **DEPTH Rounds**  | 5 (creative iteration)   |
| **Framework**     | VIBE (not RCAF/COSTAR)   |
| **Scoring**       | EVOKE (not CLEAR)        |
| **Threshold**     | 40+/50                   |
| **Output Length** | 100-300 words            |

| **K**inetic   | 10     | 8+        | Suggests motion and life?       |
| **E**motional | 5      | 4+        | Conveys experiential goals?     |

### Scoring Signals

**Evocative (+):**
- Sensory language present
- Metaphors and analogies used
- Abstract concepts invoked
- Emotional textures described

**Evocative (-):**
- Literal descriptions only
- Technical jargon present
- Specification-heavy language

**Visual (+):**
- Color mood described
- Spatial relationships implied
- Visual metaphors used

**Visual (-):**
- Vague terms ("nice", "good")
- Contradictory elements

**Open (+):**
- Suggestive rather than prescriptive
- Multiple valid interpretations possible
- Direction without destination

**Open (-):**
- Overly specific constraints
- Implementation prescriptions

**Kinetic (+):**
- Interaction feelings described
- State transitions mentioned
- Dynamic qualities present

**Kinetic (-):**
- Static-only description
- Mechanical language

**Emotional (+):**
- User feelings articulated
- Mood atmosphere defined
- Experiential outcomes stated

**Emotional (-):**
- Emotionally neutral language

### Score Interpretation

| Score | Assessment   | Action                        |
| ----- | ------------ | ----------------------------- |
| 45-50 | Excellent    | Output immediately            |
| 40-44 | Good         | Output with minor suggestions |
| 35-39 | Adequate     | Offer refinement options      |
| 30-34 | Weak         | Require iteration             |
| 0-29  | Insufficient | Block, request more input     |

| Aspect     | EVOKE                           | CLEAR                       |

### 6.1 Screenshot Analysis Protocol

> **Core Principle:** Feel first, analyze second. Capture the emotional response before breaking down elements. Describe, don't measure.

When a user provides a screenshot, image, or visual reference, extract evocative context using this protocol:

#### Phase 1: First Impression (3-Second Rule)

Before analyzing details, capture the immediate gut response:

| Question | Purpose |
|----------|---------|
| What's the **overall vibe**? | Single word or phrase (calm, energetic, premium, playful) |
| What **emotion** hits first? | Trust, excitement, focus, delight, calm, urgency |
| What's the **dominant characteristic**? | Whitespace, color, typography, density, imagery |

```
CAPTURE IMMEDIATELY:
"This feels like... [one sentence]"
"The first thing I notice is... [dominant element]"
"It makes me feel... [emotion]"
```

#### Phase 2: Visual VIBE Extraction

Systematically extract each VIBE element from what you observe:

**V â€” Vision (What does it look like?)**
| Observe | Ask Yourself | Extract |
|---------|--------------|---------|
| Overall aesthetic | Modern or classic? Minimal or rich? | Style keywords |
| Spatial feeling | Tight and efficient or airy and spacious? | Spatial vocabulary |
| Color impression | Warm or cool? Muted or vibrant? Monochrome or colorful? | Color mood |
| Surface treatment | Flat or layered? Sharp or soft? | Depth vocabulary |

**I â€” Inspiration (What does it remind you of?)**
| Observe | Ask Yourself | Extract |
