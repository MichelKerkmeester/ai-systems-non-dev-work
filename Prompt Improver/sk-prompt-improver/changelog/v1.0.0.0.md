## [**1.0.0.0**] - 2026-06-10

Prompt Improver adopts the dual-runtime packaging model for the first time. Before this release the system shipped as a single versioned knowledge base, files like `Prompt - System - Prompt - v0.200.md` and `Prompt - Templates - Visual Mode - v0.200.md` loaded whole with no router in front of them. This release introduces `skill/SKILL.md` at version 1.0.0 as a CLI source of truth, ten paired `skill/references/*.md` files, a hand-synthesized `claude project/Custom Instructions.md` plus `claude project/README.md` for the derived claude.ai Project and a new `SYNC.md` manifest defining the contract between the two packagings for the first time. Version records for this system begin here. The change lands as part of a same-day rollout across all ten Barter AI systems, described in the commit message as the shared "skill + claude-project packaging model."

> Scope: packaging model introduction. First SKILL.md-based routing for this system, replacing the single-file knowledge base.

---

### skill/ And claude project/ Split

`skill/SKILL.md` ships at 297 lines and version 1.0.0, carrying identity, activation triggers, routing and rules that previously lived only in the standalone system prompt. Ten paired reference files land under `skill/references/`: `depth_framework.md`, `format_guide_json.md`, `format_guide_markdown.md`, `format_guide_yaml.md`, `image_mode.md`, `interactive_mode.md`, `patterns_evaluation.md`, `system_prompt.md`, `video_mode.md` and `visual_mode.md`. Each pairs with a matching `claude project/knowledge/` mirror under a descriptive filename, for example `Prompt Improver - DEPTH Thinking Framework - v0.200.md`, and the new `claude project/Custom Instructions.md` (134 lines) synthesizes the same rules for claude.ai's no-filesystem environment.

&nbsp;

### SYNC.md Manifest

`SYNC.md` did not exist before this release. The new 36-line manifest states the sync rule for the first time: `skill/references/` is canonical, Project Knowledge mirrors regenerate by byte-for-byte copy into `claude project/knowledge/` with paired display filenames and a drift-audit shell sequence (symlink check, sha256 checksum, word count, stale-name grep) closes the file.

&nbsp;

### Root Doc Hygiene

`AGENTS.md` collapses from 233 lines to 31, dropping the earlier persona-and-boundaries prose in favor of a short enforcement wrapper that hands identity straight to `skill/SKILL.md`. `README.md` loses its numbered table of contents and decorative emoji section markers, trimming from 927 to 911 lines.

&nbsp;

### Legacy Knowledge Base

The pre-existing versioned knowledge base, ten files spanning `Prompt - Reference - Format Guide JSON - v0.140.md` through the older `v0.121` template files, is archived under `z_archive/knowledge base/` for manual comparison rather than deleted.
