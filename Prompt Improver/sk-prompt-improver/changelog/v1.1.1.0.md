## [**1.1.1.0**] - 2026-06-15

Layout flatten and fleet-wide OVERVIEW normalization. SKILL.md's frontmatter version reads 1.1.1 at this commit, up from the 1.0.0 the June 10 rollout shipped. No intermediate 1.1.0 commit exists in this folder's history, so this entry documents the state actually on disk rather than an inferred stepping stone. The commit message describes the wider fleet work as a coordinated normalization pass: every real skill reference and asset across all ten systems opens with a normalized `## 1. OVERVIEW` section, and Custom Instructions files get a prose pass removing semicolons and em dashes. For Prompt Improver specifically, this release removes the standalone system-prompt reference, splits the three largest mode references into slim workflow references plus new library assets and renames the `z_archive/knowledge base/` folder to `z_legacy/`.

> Scope: reference extraction, format-guide relocation and archive rename. No scoring-gate change.

---

### System Prompt Retirement

`skill/references/system_prompt.md`, 683 lines, is deleted outright. The new SKILL.md states directly that it is now "the single operating brain" for the system and that "no companion system-prompt file is required or allowed."

&nbsp;

### Mode Library Extraction

Three oversized mode references shrink by moving their example banks and quick-lookup material into new sibling asset files. `image_mode.md` drops from 1316 to 505 lines as the new `assets/image_mode_library.md` (1517 lines) takes the extracted material. `video_mode.md` drops from 1072 to 328 lines against a new `assets/video_mode_library.md` (1170 lines). `patterns_evaluation.md` drops from 886 to 665 lines as its framework matrix and selection algorithms move into a new `assets/framework_pattern_library.md` (588 lines). `visual_mode.md` shrinks more modestly, from 1209 to 1014 lines, alongside a new `assets/visual_mode_library.md` (476 lines).

&nbsp;

### Format Guides Relocate

`format_guide_json.md`, `format_guide_markdown.md` and `format_guide_yaml.md` move from `skill/references/` to `skill/assets/`, a pure path rename with the content otherwise unchanged.

&nbsp;

### Archive Rename

`z_archive/knowledge base/{reference,system,templates}/`, the ten files archived at the June 10 rollout, renames flat to `z_legacy/` with zero content change per file. The paired `claude project/knowledge/` mirrors pick up their own version bumps in the same pass: Patterns and Evaluation moves v0.201 to v0.210, Visual Mode moves v0.200 to v0.201, the three Format Guide mirrors move v0.140 to v0.141 and Image Mode plus Video Mode split into new "Assets - Image Mode Library" and "Assets - Video Mode Library" mirrors at v0.101.
