## [**1.1.2.0**] - 2026-06-21

Visual Mode grounding-first reconciliation, shipped in an explicit "Prompt Improver v1.1.2" commit. The release reconciles Visual Mode (`$vibe`/`$v`) with an anti-default design canon: briefs must ground their subject before the VIBE framework runs, and the EVOKE scoring gate rejects templated output regardless of how evocative it reads. This is a scoring-surface change, the first since the June 15 flatten, not a packaging change.

> Scope: Visual Mode workflow and EVOKE scoring. SKILL.md version 1.1.1 to 1.1.2.

---

### Step 0 Grounding And The Anti-Default Gate

Visual Mode gains a non-negotiable "Ground the Subject" step, covering Subject, Audience and Single Job, that runs before VIBE. Emitted briefs now carry a named avoidance list (generic SaaS gradient, centered hero plus three cards, untouched component-library surface, three AI-default clusters) plus one justified aesthetic risk. The EVOKE grounding pre-check scores a brief 0 regardless of evocative quality if it lacks subject grounding or reads as a templated default, while the underlying 50-point dimension weights stay unchanged.

&nbsp;

### Eight Category Defaults, Not A Menu

The eight design directions (Precision, Warmth, Sophistication, Boldness, Utility, Data, Journey and Flow, Narrative and Story) are reframed from a pick-one menu into category defaults to name and then deliberately deviate from. Reusable, across-briefs style presets are now prohibited. A pre-output critique gate asks "would I produce this for any similar brief," and revises the brief if the answer is yes.

&nbsp;

### File And Version Changes

`skill/SKILL.md` moves from version 1.1.1 to 1.1.2 to reflect the updated Visual Mode summary line. `skill/references/visual_mode.md` and the paired `Prompt Improver - Visual Mode - v0.300.md` mirror carry the bulk of the rewrite, 324 lines changed, with the mirror's version jumping from v0.201 to v0.300 to signal the redesign. `patterns_evaluation.md` and its mirror move from v0.210 to v0.211 with a 32-line change adding a seed-of-thought debias note for N of 2 or more variations. `README.md` gains a new "MAJOR UPDATE: Visual Mode Grounding-First Reconciliation" section documenting the same list of changes for CLI readers, and `claude project/Custom Instructions.md` and `claude project/README.md` pick up matching version-stamp updates.
