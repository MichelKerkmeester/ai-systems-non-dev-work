## [**1.0.0.0**] - 2026-06-10

Founding release. Product Owner received its first `skill/` plus `claude project/` split in the system-wide packaging migration, with `skill/SKILL.md` born at version 1.0.0 (218 lines) beside the legacy `Product Owner - System - Prompt - v0.997.md` identity file it would replace five days later.

> Scope: the founding release of the dual-runtime packaging model this system still uses.

---

### The Two-Packaging Model

The system-wide packaging migration commit gave every Barter AI system the same split on this date: a `skill/` folder as the source of truth and CLI runtime identity, entered through an `AGENTS.md` bootstrap, beside a `claude project/` folder carrying a hand-synthesized kernel and knowledge mirrors for claude.ai. Product Owner lived at the repository root already, so the flatten that moved other systems did not move it.

&nbsp;

### Version Anchor

`skill/SKILL.md` was pinned at `version: 1.0.0` in this commit, read directly from the historical frontmatter. Later releases in this changelog build on that pin.
